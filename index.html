AniFlix adalah aplikasi client-side yang melakukan crawling pada website publik. Kami tidak menghosting, mengunggah, atau menyimpan file video/gambar apapun di server kami.

Aplikasi ini dibuat semata-mata untuk tujuan pendidikan pengembangan perangkat lunak. Pengembang tidak bertanggung jawab atas penggunaan aplikasi ini oleh pengguna akhir. Jika Anda pemegang hak cipta, silakan hubungi sumber asli konten atau kontak kami untuk masalah teknis repositori.

Kami sangat menyarankan pengguna untuk mendukung industri anime dengan menonton melalui platform resmi berlisensi.
