Nonton Anime dan Baca Komik Tanpa Batas

Proyek open-source yang dirancang untuk memudahkan menonton anime dan membaca komik - gratis sepenuhnya, tanpa iklan, tanpa ribet.

Unduh Sekarang Lihat di GitHub

Keunggulan AniFlix

Open Source

Dibangun dengan React Native dan TypeScript. Bebas menggunakan kode kami untuk membuat aplikasi Anda sendiri di bawah lisensi GPLv3.

Koleksi Lengkap

Akses Ribuan anime/movie/komik(manga,manhwa,manhua) secara gratis tanpa iklan.

Update Berkala

Pembaruan aplikasi dan konten secara rutin untuk pengalaman menonton terbaik.

Mudah Digunakan

Antarmuka sederhana dan intuitif. Unduh episode dengan satu klik. Tidak perlu login.

Penting!

Semua video dan data apapun yang tersedia di aplikasi tidak dihosting di server kami. Mereka berasal dari sumber pihak ketiga dan kami tidak memiliki kendali atasnya.

Kami hanya membagikan video dan gambar yang tersedia secara publik di internet dan tidak menyimpan konten apa pun di server kami sendiri.
